# 💊 Java Pharmacy Management System

A desktop-based Pharmacy Management System built using Java & Swing. This application helps manage medicines, billing, and pharmacy operations efficiently.

---

## 🚀 Features

* 📦 Medicine Management (Add, Update, Delete)
* 🧾 Billing System
* 📊 Dashboard Overview
* 🔍 Search functionality

---

## 🛠️ Tech Stack

* Java
* Swing (GUI)

---

## ▶️ How to Run

1. Clone the repository
2. Open in NetBeans / IntelliJ
3. Run the Main.java file

---

## 📂 Project Structure

* `src/` → Source Code
* `resources/` → Images & assets
* `nbproject/` → NetBeans config
